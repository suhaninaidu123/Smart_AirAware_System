# Smart_AirAware_System
AirAware is a smart environmental monitoring and prediction system designed to analyze historical air pollution data and forecast future Air Quality Index (AQI) levels. The system helps users understand pollution trends and take preventive actions by providing accurate air quality predictions through a user-friendly dashboard.

The project integrates Machine Learning, Data Analysis, and Web Technologies to build an intelligent platform that predicts AQI based on historical pollutant data such as PM2.5, PM10, NO₂, SO₂, CO, and O₃ levels. The prediction model is trained on historical air quality datasets to identify patterns and forecast upcoming pollution levels.

AirAware includes an interactive interface where users can view AQI predictions, pollution trends, and environmental insights. The system processes the input data, runs it through a trained machine learning model, and displays predicted AQI values in real time.

# Key Features
1. AQI prediction using Machine Learning models

2. Analysis of historical air pollution data

3. Interactive dashboard for pollution monitoring

4. User-friendly interface for viewing AQI forecasts

5. Data visualization for pollution trends
